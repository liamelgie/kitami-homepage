Things to note

Transcoding

Transcoding (converting one file format to another) uses A LOT of processing power and is a nightmare when streaming content to a lot of users at once. However, by streaming at the highest quality possible (referred to as 'Original Quality' in Plex), the server can handle many streams without issue.

The Plex web app is limited by the file formats supported by the browser it's running in. Where possible, I use HEVC/x265 due to its many benefits but browsers currently lack support for this standard. For best results, use the Plex Media Player app when streaming content on your PC.

On smartphones/tablets or TVs, just use the native app for that device and make sure that it's streaming at 'Original Quality'. If you're using iOS or tvOS, I highly recommend Infuse as it can directly play content extremely quickly and looks beautiful doing it.
